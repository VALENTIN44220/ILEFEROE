Jours 6 à 8 — recommandations par journée

Jour 6 — Mercredi 17 juin • Kalsoy (Kallur) depuis Klaksvík
Zone : Klaksvík / Kalsoy

Déjeuner (pratique)

  • Avant le ferry : cafés/snacks près du port (selon horaires).
  • Sur Kalsoy : prévoir un pique-nique (options limitées).

Dîner (Klaksvík)

  • Option simple : restaurant de l’hébergement (ex : Hotel Klaksvík).
  • Port / centre : recherche restaurants Klaksvík (Google Maps)

Spécialités

  • Poisson (frais), soupe de poisson, agneau (si proposé).

Conseils

  • Vérifier les horaires ferry : journée rythmée → mieux vaut réserver/repérer une option dîner à l’avance.
  • Emporter eau + snacks pour la rando (vent + froid).
Jour 7 — Jeudi 18 juin • Trælanípa + soirée près de l’aéroport
Zone : Vágar

Déjeuner

  • Avant/Après rando : options autour de Miðvágur / Sandavágur.
  • cafés Sandavágur (Google Maps)

Dîner (clôture)

  • Option confortable : restaurant d’hôtel / guesthouse.
  • Sinon : restaurants Sørvágur (Google Maps)

Idée “local”

  • Poisson (cabillaud/saumon), agneau, desserts nordiques simples.

Conseils

  • Après Trælanípa : vous serez fatigués → privilégier une option dîner proche de l’hébergement.
  • Si météo très ventée : déplacer Trælanípa plus tôt et garder un “plan B” (balades villages).
Jour 8 — Vendredi 19 juin • Départ (vol 12:20)
Zone : Vágar / aéroport

Petit-déjeuner

  • Priorité : petit-déj à l’hébergement (gain de temps).
  • Alternative : petit-déj proche aéroport (Google Maps)

Conseils

  • Prévoyez une marge pour restitution voiture + carburant.
  • Limiter le programme le matin : pas de rando, uniquement trajets courts.
